Glitches & Guides

1. How to Apply Crew Colors ▼
1. Go to Rockstar Social Club -> Edit Crew.
2. Right-click the color box -> Inspect Element.
3. Find the value ending in #.......
4. Paste your Hex. If "Not Supported", add 00 or AA to the end.
5. Save Crew and restart game.
2. Matte + Pearl Glitch ▼
1. Go to LSC. Apply Matte color.
2. Go to Metallic, scroll 3s.
3. Back out, buy Plate.
4. Go back to Respray > Pearlescent.
3. Pure Black Smoke ▼
1. Set Crew Color to #000000AA.
2. Buy Patriot Tire Smoke.
3. Change Smoke to Crew Smoke.
4. Instant Update ▼
1. Save color on Social Club.
2. Switch from Online to Story Mode.
3. Return to Invite Only Session.
4. Color is now updated in LSC.
5. Secret Gold ▼
Use Hex #B58A40 over Brushed Gold or Pure Gold metals for a hidden texture.
