Glitches & Guides

1. How to Apply Crew Colors ▼
1. Go to Rockstar Social Club -> Edit Crew.
2. Right-click the color box -> Inspect Element.
3. Find the value ending in #.......
4. Paste your Hex. If it says "Not Supported", add 00 or AA to the end (e.g., #FF0000AA).
5. Save Crew and restart game.
2. Matte + Pearlescent Glitch ▼
1. Go to LSC. Apply a Matte color.
2. Go to Metallic, scroll for 3 seconds (don't buy).
3. Back out, buy a License Plate (change text/color).
4. Go back to Respray > Pearlescent.
5. It is now unlocked on Matte paint.
3. Pure Black Smoke (Patriot) ▼
1. Set Crew Color to #000000AA.
2. Buy Patriot Tire Smoke in LSC.
3. Change Smoke to Crew Smoke.
4. The smoke will now be pitch black instead of Patriot colors.
4. Instant Crew Color Update ▼
You don't need to restart the whole game!
1. Change color on Social Club & Save.
2. In Game: Switch from Online to Story Mode.
3. Once in Story Mode, go back to Invite Only Session.
4. The color is now updated in LSC.
5. Secret Gold Paint ▼
This hex mimics the "Secret Gold" texture often found on modded cars.
Hex: #B58A40
Apply this over "Brushed Gold" or "Pure Gold" metals for different effects.

About GTA Crew Tool

The ultimate utility for customizing vehicle colors in GTA Online. Created by ArtfulTwo79049.

Features

  • Hex Editor: Fine-tune RGB values and generate specific hex codes.
  • OEM Library: Over 100+ real-world paint codes from Ferrari, Porsche, JDM legends, and more.
  • Visualizer: Preview colors with accurate Pearlescent overlays before applying them in-game.
  • Glitch Guides: Built-in tutorials for the most popular paint glitches.
  • Offline Capable: Install as a PWA on iOS and Android.

How To Use

  • Create: Use the sliders or library to select a color.
  • Visualize: Select a "Pearlescent" from the dropdown to see how it affects the base coat.
  • Copy: Hit the "Copy Hex" button.
  • Apply: Go to Social Club > Edit Crew > Inspect Element > Paste Hex > Save.
This tool is fan-made and not affiliated with Rockstar Games.
